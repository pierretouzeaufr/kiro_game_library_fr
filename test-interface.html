  3. Les liens "API REST" retournent du JSON brut
  4. L'interface des alertes est à : http://localhost:8081/alerts

🚨 Interface des Alertes

L'interface des alertes comprend :

  • ✅ Liste des alertes actives avec icônes colorées
  • ✅ Formulaire de création d'alertes personnalisées
  • ✅ Actions rapides (génération automatique, nettoyage)
  • ✅ Boutons pour marquer comme lue ou supprimer
  • ✅ Affichage des noms complets (utilisateurs et jeux)

URL directe : http://localhost:8081/alerts
